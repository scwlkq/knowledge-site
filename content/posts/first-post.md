# 欢迎

这是我的第一篇文章，使用 Hugo 搭建的个人知识站。

## 特点

- 简洁的设计
- Markdown 写作
- Git 提交自动发布

## 使用方法

1. 在 `content/posts/` 目录下创建新的 `.md` 文件
2. 编写 Markdown 内容
3. `git add` 和 `git commit`
4. `git push` 到 GitHub
5. 自动部署到线上

就这么简单！
